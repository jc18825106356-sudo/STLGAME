# Godot 4.6 AI 提问模板合集

这是为 Godot 4.6 游戏开发专门整理的 AI 提问模板包，包含基础、调试、逻辑设计、优化、学习以及进阶版模板，可直接用于向 LLM 提问生成 GDScript 或设计方案。

---

## 一、基础 API 使用模板

**模板结构**：
```
项目类型: [2D/3D 游戏]
目标: 实现 [功能]
限制: 使用 Godot 4.6
输出: GDScript 示例代码 + 每行注释
```

**例子 1：玩家移动**
```
项目类型: 2D 平台游戏
目标: 玩家左右移动并跳跃
限制: Godot 4.6
输出: GDScript 示例代码，并解释 move_and_slide 与 Input 的关系
```

**例子 2：敌人巡逻**
```
项目类型: 2D 动作游戏
目标: 敌人在平台上左右巡逻
限制: Godot 4.6
输出: GDScript 示例代码 + 节点设置说明
```

**例子 3：投篮功能**
```
项目类型: 2D 投篮小游戏
目标: 玩家投篮并计算得分
限制: Godot 4.6
输出: GDScript 示例代码 + 注释每行逻辑 + 处理碰撞检测
```
